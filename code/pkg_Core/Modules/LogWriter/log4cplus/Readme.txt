Logging Framework for C++
http://sourceforge.net/projects/log4cplus

Download from http://sourceforge.net/projects/log4cplus (eg: log4cplus-1.0.4-rc11).
Copy it's folders (only need 'include' and 'src') to LogWriter\log4cplus\.

log4cplus-1.0.4 has some compiler errors for VC++ 6.0, you can find compatible patch about it.
