pkg_UnitTest: unittest projects, using cppunit 1.12.1.
http://sourceforge.net/projects/cppunit/files/cppunit/1.12.1/

Download cppunit include files and copy to this 'cppunit' folder:
    'pkg_UnitTest\Interface\cppunit\'.
    
Copy cppunit's LIBs and DLLs to folder 'code\bin\vcxx\xxx\unittests'.
方正内网中有各个版本的LIB和DLL: \\dampub\Tools\CodeCheck\C++\CppUnit\cppunit-1.12.1\lib\

Running unittest application will create the config file 'UnitTests.ini'.
